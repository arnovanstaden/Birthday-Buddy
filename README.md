# Birthday Buddy
A lightweight PWA for easy birthday reminders & congratulatory message sending.

## Link
You can visit the site [here](birthday-buddy.vercel.app).

## Features
* Save Birthdays, Keep Track of Upcoming Birthdays & Receive/Set Reminders
* Pre-fill & Share Personalised Messages via Your Preferred App.
* Authentication to Persist Birthdays Across Devices
* Mobile-First Native App Design & Experience
* Progressive Web App Functionality: Native Install & Push Notifications
