# Birthday Buddy
A lightweight PWA for easy birthday reminders, sharing & congratulatory message sending.

## Link
You Can Explore This App With A Demo Account [here](https://birthday-buddy.vercel.app/signin?test).

## Features
* Save Birthdays & Keep Track of Upcoming Birthdays
* Receive & Set Birthday Reminders
* Share Birthdays with Friends to Save on Their Account
* Authentication to Persist Birthdays Across Devices
* Mobile-First Native App Design & Experience
* Progressive Web App Functionality: Native Install & Push Notifications
