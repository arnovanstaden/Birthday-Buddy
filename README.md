# Birthday Buddy
A lightweight PWA for easy birthday reminders & congratulatory message sending.

## Link
You can visit the site [here](https://birthday-buddy.vercel.app/).

## Features
* Authentication
* Mobile-First Native App Design & Experience
* Persistent User Data Across Devices
* Progressive Web App Functionality
* Import Data from Native Contacts
* Send Pre-written Messages with Multiple Social Apps
* Push Notifications & Email Reminders

## Additional Information

### Type
Progressive SPA PWA
